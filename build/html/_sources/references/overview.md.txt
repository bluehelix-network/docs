# BHChain API


<a name="overview"></a>
## Overview
A REST interface for state queries, transaction generation and broadcasting.


### Version information
*Version* : 1.0


### URI scheme
*Host* : stargate.cosmos.network  
*Schemes* : HTTPS


### Tags

* Cu : Custodian unit module APIs
* Distribution : Fee distribution module APIs
* Governance : Governance module APIs
* Mint : Minting module APIs
* Misc : Query app version
* Slashing : Slashing module APIs
* Staking : Stake module APIs
* Supply : Supply module APIs
* Tendermint RPC : Tendermint APIs, such as query blocks, transactions and validatorset
* Transactions : Search, encode, or broadcast transactions.
